# Portfolio Analysis, Asset Allocation, and Optimization Using Monte Carlo Simulation

This project applies Modern Portfolio Theory (MPT) to construct an optimal investment portfolio using Python. It includes return calculations, risk measurement, Monte Carlo simulations, and optimization techniques to identify the best risk-return trade-off.

- Daily return computation for selected assets  
- Randomized portfolio generation

- Optimal portfolio identification (maximum Sharpe & minimum variance)
