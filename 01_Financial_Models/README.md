# Financial Models

This folder contains 3-statement and DCF models. Each Excel workbook is accompanied by:
- A brief README (one-liner)
- Assumption sheet with sources
- Output summary (intrinsic value & sensitivity table)

**Files**
- `MRF_3Statement_Model.xlsx` - 3-statement linked model and sensitivity
- `Infosys_DCF_Model.xlsx` - DCF model (Revenue, WACC, TV, Sensitivity)
### Infosys DCF Valuation Model
This model includes 5-year historical data, 5-year projections, FCFF estimation, WACC, CAPM analysis, terminal valuation (Gordon & Exit Multiple), and a sensitivity table. Intrinsic value is computed based on the discounted FCFF approach.
